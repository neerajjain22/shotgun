# seo-copywriting-guidelines.md

## Purpose
Defines how Shotgun website copy should be written for SEO performance and clear communication.

Primary rule: **Clarity > Cleverness**.

## Ideal Customer Persona (ICP)
Write for:
- Shopify founders
- Ecommerce operators
- Growth and marketing teams
- Product managers working on Shopify stores

Audience traits:
- Busy
- Operational
- Outcome-focused
- Low tolerance for jargon

## Core Copy Principle
Use direct language that explains value quickly.

Avoid clever slogans that hide meaning.

Example:
- Weak: "Reinvent the way commerce moves."
- Strong: "Get Shopify development done instantly."

## Language Style Rules
Use:
- Simple language
- Short sentences
- Active voice
- Concrete examples

Avoid:
- Buzzwords
- Vague claims
- Abstract language
- Clever headlines that reduce clarity

## SEO Writing Principles
Every page must map to a clear search intent.

Each page should include:
- One H1 with the primary keyword
- H2 sections for related subtopics
- Internal links to relevant pages

Metadata baseline:
- Title tag
- Meta description

URL baseline:
- Descriptive, keyword-based, lowercase kebab-case paths

## Keyword Strategy (Shotgun Focus)
Prioritize query families around:
- Shopify development
- Shopify theme changes
- Shopify store optimization
- Shopify developer help
- Shopify website fixes

Rule:
- Target real operator problems and actions, not vanity keywords.

## Content Structure Rules
Preferred page pattern:
1. Clear headline
2. Short explanation
3. Scannable bullets and/or short sections
4. Concrete examples
5. Action-oriented CTA

Avoid dense walls of text.

## Concrete Example Requirement
When relevant, include practical examples such as:
- Make a homepage banner clickable
- Add personalization fields to a product page
- Launch a weekend sale with discount codes
- Fix mobile layout issues

Examples should help visitors immediately understand what Shotgun can execute.

## Mandatory Page Questions
Every page should answer quickly:
1. What does Shotgun do?
2. Who is it for?
3. What problem does it solve?
4. What outcome does the user get?

## Messaging Quality Checklist
Before publishing or merging copy changes, confirm:
- The primary keyword and intent are explicit.
- Headline is clear and specific.
- Copy speaks to Shopify operators, not generic audiences.
- Claims are concrete and believable.
- At least one concrete use-case example is included when applicable.
- Internal links point to related product/use-case/resource pages.
- Metadata is present and aligned with on-page message.

## Maintenance Rule
When creating or editing website copy:
1. Review this document and `SEO_RULES.md` first.
2. Ensure clarity, intent-match, and ICP relevance.
3. Update this guideline if copy standards materially evolve.
